# Spain Route Search using AI Algorithms

This project implements classical Artificial Intelligence search algorithms to find the shortest path between cities in Spain.

## Algorithms Implemented
- Breadth-First Search (BFS)
- Greedy Best-First Search (GBFS)
- A* Search

## Heuristics Used
- Haversine Distance
- Euclidean Distance

The Haversine heuristic estimates the great-circle distance between two cities based on latitude and longitude.

## Objective
Compare the performance of different search algorithms in solving real-world pathfinding problems.

## Results
A* Search with Haversine distance produced the most optimal and efficient path compared to BFS and Greedy Best-First Search.

## Technologies
- Python
- NumPy
- Matplotlib
- Graph-based search algorithms

## Example Visualization
The project visualizes routes between Spanish cities such as Madrid and Barcelona.
